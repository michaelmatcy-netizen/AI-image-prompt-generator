# 🎨 AI Image Prompt Generator Bot

A Telegram bot that generates images from text descriptions using AI.

## ✨ Features

- Generate images from text prompts
- Customize image size (512x512, 768x768, 1024x1024)
- Choose from 8 different art styles
- Interactive buttons for easy navigation
- Example prompts to get started
- Pro tips for better results

## 🚀 Commands

- `/start` - Start the bot
- `/help` - Show help message
- `/settings` - Customize size and style
- `/examples` - See example prompts
- `/reset` - Reset to default settings

## 🛠️ Setup

1. Get a Telegram Bot Token from @BotFather
2. Clone this repository
3. Install dependencies: `pip install -r requirements.txt`
4. Create `.env` file with your BOT_TOKEN
5. Run: `python bot.py`

## 📦 Deployment

Deploy easily on Railway:
1. Push to GitHub
2. Connect Railway to your repo
3. Add BOT_TOKEN in Railway variables
4. Deploy!
